# B205 Chat Application

## Project Overview

This project is a socket-based messaging application developed for the B205 Computer Networks module.

### Features
- Multi-user chat
- Multiple chat rooms
- User invitations
- Real-time messaging
- TCP socket communication
- Configuration management
- Logging
- Error handling

### Technology
- Python
- TCP Sockets
- JSON Protocol
- Multithreading
